# Etnos


## Introdução

Repositório destinado ao prototipo da aplicação para o Cliente.

Iremos usar o [Node Web Kit](https://github.com/rogerwang/node-webkit) e [NodeJS](https://github.com/joyent/node) para desenvolver a aplicação.

## Como executar

```
$ npm start
```

## Build

```
$ npm install
$ npm install gulp -g
$ gulp dist

```
